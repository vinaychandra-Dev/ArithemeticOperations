# ArithmeticOperations

A simple Java console application that performs basic arithmetic operations:  
✅ Addition  
✅ Subtraction  
✅ Multiplication  
✅ Division (with divisor zero check and floating-point result)  

The program uses a loop to allow multiple calculations until the user quits by pressing `Q` or `q`.

---

## 📌 Features
- Handles integer inputs
- Provides clean output messages
- Checks for division by zero
- Allows repeated calculations without restarting
